## Overview
Sometimes our services exchange or process information with legacy systems and that takes time. Having a spinner lets user know that their request is being worked on.

### How the component is currently used

The component is a screen pattern that acts as a progress indicator between 2 screens.
Example: When the submit button is clicked on the final screen to send details, it redirects user to the spinner screen and then return the result - success or failure.
