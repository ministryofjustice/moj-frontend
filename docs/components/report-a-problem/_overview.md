## Overview
Collapsible support component that helps users report tech-related page issues. Displays technical diagnostic information that users can provide when contacting support, alongside guidance on how to raise an issue. Vastly helps with diagnosing and resolving issues for users.

### How the component is currently used

It is being used in the new ARNS Assessment Platform / Assess and Plan service, with it currently intended to be released alongside the Sentence Plan build in late March. Sits neatly on top of the footer.

### Contribute to this component
You can help develop this component by adding information to the [‘report a problem’ Github discussion]({{ githuburl }}). This helps other people to use it in their service.
