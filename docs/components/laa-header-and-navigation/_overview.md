## Overview
How it works:

The LAA header tells users they are using a service from the Legal Aid Agency. The header displays the Legal Aid Agency crest and organisation name, and displays who is signed in with a link to sign out.

The pattern also includes the service name, service navigation with a link to change service (if applicable), phase banner, back link and English/Welsh language toggle.

When to use this component:

The LAA header is for internal services used by legal aid providers and caseworkers.

LAA UCD teams collectively agreed to class 'providers' as non-public users because they need permission to access LAA services.

Certain LAA services should continue to use the blue GOV.UK header if they are public facing, i.e:
• Check if you can get legal aid
• Find a legal aid adviser or family mediator

### How the component is currently used

Design thinking:

The LAA header is based on the black MoJ header designed for internal staff.

The service name is displayed using the GOV.UK service navigation pattern.

Research on this component:

No known issues - this pattern is currently in use in several live LAA services.

Known issues and gaps:

Conversation needed to agree where the user is taken after clicking the organisation name.
