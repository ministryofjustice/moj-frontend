### help content (DPS only)

When directing users to help and support, always use this content:

**Get help with DPS**  
Call 0800 917 5148 or #6598 from inside an establishment.

This content does not provide any alternative contact methods such as email or chat. This may be raised as a recommendation if your service receives an accessibility audit.

{% endaccordionSection %}
{% accordionSection "N" %}

### No results (from search)

Add the following heading after a user has conducted a search:

### 0 results for 'blah blha' in open cases. 

Offer the user help refining their search. This will vary per search but could include:

- searching under a different category, for example 'person identifier number' rather than 'name'
- guidance on the correct formats for search (which may be guided by technical constraints)
- a route to contact help or raise a ticket, if appropriate

Work to prevent errors, for example by considering validation carefully.
